## 📌 Project Overview
**LIFE Quest** is a lightweight, zero-dependency educational web application built specifically for the LIFE Program at Coastal Carolina University. By wrapping clinical social-skills training in a familiar, retro-style platformer, this tool aims to increase student engagement and provide advisors with a measurable framework for behavioral development.

The engine features a procedural 10-level architecture, taking players across recognizable CCU campus locations while presenting them with real-world social encounters.

## ✨ Core Features
* **10 CCU-Themed Levels:** Procedurally generated environments including Prince Lawn, the HTC Center, Kimbel Library, and Hicks Dining Hall.
* **Interactive Dialogue System:** A custom UI layer overlay that interrupts physics to present multiple-choice social scenarios.
* **Clinical Scoring Engine:** Tracks player choices in real-time, rewarding positive social navigation and deducting points for maladaptive responses.
* **Zero-Dependency Architecture:** A monolithic, single-file application requiring no `npm` installations or complex build steps.
* **Responsive Physics:** Powered by Phaser 3's Arcade Physics engine for smooth, snappy 2D platforming.

## 🛠️ Technical Stack
* **Game Engine:** [Phaser 3](https://phaser.io/) (via CDN)
* **UI/UX Styling:** [Tailwind CSS](https://tailwindcss.com/) (via CDN)
* **Language:** Vanilla JavaScript (ES6+), HTML5
* **Design Language:** Glassmorphism UI with CCU brand accents (Teal & Bronze)
